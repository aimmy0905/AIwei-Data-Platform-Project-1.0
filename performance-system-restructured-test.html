🚨 关键差异说明

重构前后的主要变化

!
术语纠正：将"智能方案"更正为"职能方案"（运营部门）和"过程管理指标"（销售部门）
!
部门完全分离：运营部门和销售部门使用完全不同的数据结构和考核逻辑
!
考核周期差异：运营部门项目经理季度考核，其他全部月度考核
!
评分维度不同：运营部门三维度（数据+客户+职能），销售部门两维度（月度提成+绩效底薪）
!
人工录入支持：销售部门支持客户拜访量等指标的人工录入

🏢 部门考核体系对比

🏢 运营部门考核方案

项目经理（季度考核）
考核得分 = 数据指标(40%) + 客户评价(30%) + 职能方案(30%)
数据指标：服务费目标(100%) + 返点目标(100%)
客户评价：客户满意度评分
职能方案（由项目总监评分）：
• 团队成员管理 (20%)
• 团内客户评论 (20%)
• 知识沉淀及输出 (20%)
• 培养及输出新人 (20%)
• 跨部门协助合作 (20%)
项目成员/优化师（月度考核）
考核得分 = 数据指标(40%) + 客户评价(30%) + 职能方案(30%)
数据指标：服务费目标(100%) + 返点目标(100%)
客户评价：客户满意度评分
职能方案（由项目经理评分）：
• 客户服务能力 (20%)
• 专业工作质量 (20%)
• 沟通与协助 (20%)
• 工作积极性及执行力 (20%)
• 成长与贡献 (20%)

💼 销售部门考核方案

营销经理（月度考核）
月度提成 = 数据评分(月度完成率) 100%
绩效底薪奖金 = 数据评分+过程管理指标 100%
月度提成：新单服务费目标 + 新单单量目标
绩效底薪奖金（由销售总监评分）：
• 个人新单服务费目标 (20%)
• 团队新单服务费目标 (20%)
• 团队续费服务费目标 (20%)
• 成员成长及均衡性 (10%)
• 销售流程与CRM规范 (5%)
• 策略制定与过程辅导 (10%)
• 团队客户拜访量目标 (15%) 【人工录入】
M1-M5营销顾问（月度考核）
月度提成 = 数据评分(月度完成率) 100%
绩效底薪奖金 = 数据评分+过程管理指标 100%
月度提成：新单服务费目标 + 新单单量目标
绩效底薪奖金（由营销经理评分）：
• 新单服务费目标 (50%)
• 新单客户数目标 (30%)
• 月度客户续约金额/续约服务费目标 (20%)
