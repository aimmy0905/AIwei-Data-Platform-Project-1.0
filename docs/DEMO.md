# 艾维数据平台 - 功能演示指南

## 🎯 平台概述

艾维数据平台是一个专业的数据分析管理工具，就像一个超级智能的数据助手，帮助你轻松管理和分析各种业务数据。

## 🚀 快速开始

### 1. 启动项目
```bash
# 进入项目目录
cd AIwei-Data-Platform-Project-1.0

# 启动开发服务器
npm run dev
```

### 2. 访问平台
- 打开浏览器访问：http://localhost:5175
- 使用演示账号登录：
  - 用户名：admin@aiwei.com
  - 密码：123456

### Phase 5 - 数据工具演示

#### 5.1 数据导出功能
**位置**：侧边栏 → 数据工具 → 数据导出
**功能**：
- 📥 多格式数据导出
- 📊 自定义报告生成
- 📋 导出历史管理

**演示步骤**：
1. 进入数据导出页面
2. 选择数据源和导出格式
3. 自定义导出字段
4. 生成并下载报告
5. 查看导出历史记录

**详细操作**：
```
1. 选择数据源：网站数据/广告数据/产品数据等
2. 设置时间范围：选择要导出的数据时间段
3. 选择导出格式：Excel(.xlsx)/CSV(.csv)/PDF(.pdf)
4. 自定义字段：勾选需要导出的数据字段
5. 输入文件名称：给导出文件命名
6. 点击"开始导出"：系统会显示导出进度
7. 导出完成后可以下载文件
```

#### 5.2 自定义指标设置
**位置**：侧边栏 → 数据工具 → 自定义指标
**功能**：
- 🔧 自定义指标创建
- 📊 指标模板库
- ⚙️ 计算规则配置

**演示步骤**：
1. 进入自定义指标页面
2. 创建新的自定义指标
3. 使用预设模板
4. 配置计算规则
5. 设置可视化样式
6. 保存并应用指标

**详细操作**：
```
基础配置：
1. 输入指标名称：如"转化率"
2. 添加指标描述：说明指标用途
3. 选择数据源：网站数据/广告数据等
4. 设置计算类型：求和/平均值/自定义公式
5. 选择单位：元/百分比/个数等
6. 设置数据格式：数字/货币/百分比

高级设置：
1. 选择时间维度：日/周/月/季度/年
2. 添加筛选条件：设置数据过滤规则
3. 配置排序规则：升序/降序
4. 设置显示限制：限制显示数量

可视化配置：
1. 选择图表类型：折线图/柱状图/饼图等
```

## 🎯 使用技巧

### 1. 数据分析技巧
```
看趋势：
- 关注数据的变化趋势而不是绝对值
- 结合多个指标综合分析
- 注意季节性和周期性变化

找问题：
- 使用对比功能发现异常
- 关注预警提醒信息
- 深入分析数据波动原因

做决策：
- 基于数据而不是直觉
- 考虑多个维度的数据
- 验证假设和结论
```

### 2. 报告生成技巧
```
选择合适的报告类型：
- 数据概览：全面了解业务状况
- 表现分析：深入分析特定指标
- 对比分析：比较不同时期或对象

自定义报告内容：
- 选择关键指标
- 设置合适的时间范围
- 添加必要的筛选条件
- 选择合适的可视化方式
```

### 3. 指标设置技巧
```
创建有意义的指标：
- 选择业务相关的数据源
- 设置合理的计算公式
- 使用直观的指标名称
- 添加详细的指标描述

使用预设模板：
- ROI计算：评估投资回报
- 转化率：衡量营销效果
- 客单价：分析销售情况
- 留存率：了解用户粘性
```
